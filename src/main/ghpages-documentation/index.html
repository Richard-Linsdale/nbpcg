NBPCG

NetBeans Platform Code Generator

Software and Embedded Solutions Home

A Plug-in Module for the Netbeans IDE supporting the generation of many Netbeans Platform classes and support files, using an XML based notation to describe data entities, their fields and also the node relationships of these entities.

It is a Netbeans plug-in, and works with Netbeans 8 and the Java 8 runtime environment

Current release is vx.y.zz
